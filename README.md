# IADiagn-tica-Como-a-intelig-ncia-Artificial-Analisa-Sintomas
Coleta informações > Sintomas + tempo + intensidade. Organiza os dados. Compara com um banco de padrões. Procura semelhança. Gera hióteses possíveis. Classifica urgência (leve, moderada, alta). Dá recomendações gerais (ex.:beber água, procurar médico, sinais de alerta)
